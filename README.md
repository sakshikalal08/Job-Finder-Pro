# Job Finder Pro

A responsive job search platform built using HTML, CSS, JavaScript, and REST APIs.

## Features

* Real-time job listings from external API
* Dynamic search and filtering
* Responsive UI design
* Company and location details
* Direct application links
* Modern card-based interface

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Fetch API
* Git & GitHub
